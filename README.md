# CRUD Administração de Recursos

## Instructions

**Update packages**
```
sudo apt update
```

**Install Node**
```
sudo apt install nodejs
```

**Install NPM**
```
sudo apt install npm
```

**Clone o repositório**

```
git clone https://bitbucket.org/eumagnun/mineradora-ativos-frontend
```

**Instale as dependências**

```
npm install
```
**Inicie a aplicação**
```
npm start
```
